### Week 5 Achievements:

* Completed the theoretical learning materials of **AWS First Cloud Journey – Module 03**, focusing on AWS Compute and Storage services.

* Gained a solid understanding of **Amazon EC2**, including:

  * Elasticity and automatic scaling.
  * AWS Nitro System architecture.
  * AMD and AWS Graviton processors for cost optimization.
  * Snapshots for backup and recovery.
  * Key Pair authentication.
  * User Data and Instance Metadata for automated instance initialization.

* Understood how **Application Load Balancer (ALB)** and **Auto Scaling Group** work together to improve application availability while optimizing infrastructure costs.

* Learned the characteristics and use cases of AWS pricing models:

  * On-Demand Instances.
  * Reserved Instances.
  * Spot Instances.

* Distinguished the use cases of AWS storage services:

  * Amazon EBS for persistent block storage.
  * Instance Store for high-performance temporary storage.
  * Amazon EFS for shared Linux file systems.
  * Amazon FSx for Windows and Linux workloads with data deduplication support.

* Learned the purposes of:

  * Amazon Lightsail for lightweight cloud deployments.
  * AWS Application Migration Service (AWS MGN) for application migration and disaster recovery.
