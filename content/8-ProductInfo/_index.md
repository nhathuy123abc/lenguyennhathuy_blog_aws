## Product Information: SmartCV (AI-CV AWS)

**SmartCV** is an AI-integrated application that helps users create, manage, and optimize their CVs intelligently. The project is built and deployed on AWS infrastructure, leveraging modern cloud technologies to deliver the smoothest and most optimized experience possible.

- 🔗 **Live Demo:** [https://smartcvknight.click/](https://smartcvknight.click/)
- 💻 **Source Code (GitHub):** [https://github.com/huynhnhan68/smartcv](https://github.com/huynhnhan68/smartcv)

---

### Project Highlights

- ✅ **AI Integration** — Analyzes and assists in perfecting CV content using artificial intelligence.
- ✅ **Fully deployed on AWS** — Ensures scalability and high reliability through modern cloud infrastructure.
